Solo, Self-Teaching Projects

  • Trivia Slack Bot is a node.js based app that interfaces with Slack's API. It asks a trivia question in a slack channel and records answer attempts from players within a MSSQL database. The Trivia Bot currently interacts with more than 100 players per day, posts scores for players daily, and has a small amount of conversation interaction functionality.

  • First Memories is an online journaling application for parents that want a way to record pictures and journal entries about their child from anywhere. This application uses a React/Redux framework to display user provided journal entries stored with MongoDB using Node.js and stores images with the image cloud service, Cloudinary

  • Dinner Plans is a meal preparation application designed to assist with organizing meals on a weekly basis. Using a Node.js server and jQuery front end, it provides search results using keywords from Yummly's API for recipes that can be added to the weekly view, to allow users to upload their own recipes to the weekly view, and to create a grocery list using the ingredients of the recipes in the week for ease of shopping.

  • Concert Finder is an application made to query Seat Geek's API and display search results based on user input for concerts by artist. Additionally, once found, the user can click to display a map provided by a Google Maps API query. This application, the first application I ever created, uses HTML, CSS, Javascript, and jQuery.
